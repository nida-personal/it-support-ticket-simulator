# 🖥️ IT Support Ticket Simulator

A simple IT helpdesk ticket simulation system built using **HTML, CSS, and JavaScript**.  
This project demonstrates basic IT service desk workflows such as incident logging, user input validation, and ticket display functionality.

---

🔗 Checkout Here: 👉 [View Live Project](https://nida-personal.github.io/it-support-ticket-simulator/)

---

## 📌 Project Overview

This project simulates a basic IT support ticket system where users can:
- Submit support requests
- Enter issue details
- View submitted tickets in real-time

It is designed to reflect basic **helpdesk / service desk operations** used in IT support environments.

---

## ⚙️ Features

- Create support tickets using a form
- Real-time ticket display
- Input validation (prevents empty submissions)
- Clean and simple UI design
- DOM manipulation using JavaScript
- Lightweight and fast front-end application

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 🧠 Key Learning Outcomes

- Simulating real-world IT support workflows
- Working with user input safely
